Application UsageÂ¶

Building your own MTA from the python-slimta building blocks is great, but it’s not for everyone. This guide walks you through setting up and configuring the slimta application, which glues together all the pieces with easy-to-use configuration files.

The configuration files are interpreted using the config package, which provides a great combination of convenience and readability. Config data is organized with keys (mappings) and lists, similar to JSON, with extra features like key references and file inclusion. The included sample configs use some of these features, as a reference.
