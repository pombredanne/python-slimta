slimta.redisstorage¶

This module is available in the python-slimta-redisstorage extension package.

Package implementing the queue storage system using redis.

class slimta.redisstorage.RedisStorage(host='localhost', port=6379, db=0, password=None, socket_timeout=None, prefix='slimta:')¶

Bases: slimta.queue.QueueStorage

QueueStorage mechanism that stores Envelope and queue metadata in redis hashes.

Parameters:
  • host – Hostname of the redis server to connect to.
  • port – Port to connect to.
  • db – Database number to create keys in.
  • password – Optional password to authenticate with.
  • socket_timeout – Timeout, in seconds, for socket operations. If the timeout is hit, socket.timeout is raised. None disables the timeout.
  • prefix (str) – Any key created is prefixed with this string.
